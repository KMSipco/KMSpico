How to install KMSpico

  1. Install KMSpico from one of our official download links.
  2. Go to the Start button and click on KMSpico. Click Yes if prompt User Account Control window. You will see the following screen (left). Depending on the programs installed on your PC, the icon will auto detected on the software and show the red button.
  4. If you just installed the OS from fresh, press the red button to start the activation process.
  5. If you need to check your activation status, click the Tokens tab and click the blue square with big “I” in it (right screen above). This will show your system edition as well as activation status.
  6. You also can click Start and right-click on Computer and choose Properties, the screen will show the activation status as well.

You are done and the system was successfully activated. You can enjoy the full functionalities of the system now.
